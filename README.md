# GreasyFork Premium Userscript

**GreasyFork Premium** is a userscript that modernizes and improves the GreasyFork interface to make daily usage more fluid, more readable, and more efficient.

The script focuses exclusively on the interface and user experience, without altering GreasyFork’s internal behavior.

---

## What GreasyFork Premium does

- Modernizes the GreasyFork interface
- Improves overall readability and page design
- Adds interface features designed for regular developer usage

Everything works directly on GreasyFork, without page reloads.

---

## Main features

### Interface

- Light and dark themes
- Cleaner and more consistent interface
- Improved spacing and alignment
- **Removal of link underlines**

### Navigation and quick actions

- **Script icons display** for immediate visual identification  
- **Settings and script edit buttons added directly to the page**  
- **Faster access to essential actions**, without unnecessary navigation

### Code readability

- Improved presentation of script code blocks
- Syntax highlighting via **Highlight.js**
- Better readability of scripts and code snippets
