# Contributing to GreasyFork Premium

First off, thank you for considering contributing to GreasyFork Premium.

Whether you're reporting a bug, suggesting an idea, improving documentation, or submitting code, every contribution helps improve the project.

## Ways to Contribute

You can help by:

* Reporting bugs
* Suggesting new features
* Improving documentation
* Improving translations
* Submitting code improvements

## Development Setup

1. Fork the repository
2. Clone your fork
3. Make your changes
4. Verify that your changes work as expected
5. Submit a Pull Request

## Pull Request Process

Before submitting a Pull Request:

1. Ensure your changes are tested.
2. Update documentation if necessary.
3. Keep commits focused and descriptive.
4. Link related issues when applicable.

## Code Style

* Use modern JavaScript (ES6+)
* Keep code readable and maintainable
* Use meaningful variable names
* Follow the existing project structure
* Write comments in English when needed

## Community

Please remain respectful and constructive when interacting with other contributors.

Feedback, ideas, and improvements are always welcome.

Thank you for helping make GreasyFork Premium better.
